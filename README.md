Class Mahasiswa

Class Mahasiswa digunakan sebagai blueprint/kerangka untuk membuat objek mahasiswa.
Class ini berisi atribut, konstruktor, setter-getter, dan method tampilan.

Atribut dalam class Mahasiswa

Atribut ini mewakili data yang dimiliki oleh seorang mahasiswa:

nama → tipe String
Menyimpan nama mahasiswa.

nim → tipe String
Menyimpan nomor induk mahasiswa.

jurusan → tipe String
Menyimpan jurusan atau program studi mahasiswa.

Ketiga atribut ini bersifat private, agar hanya bisa diakses melalui setter dan getter.

Fungsi class Mahasiswa

Menyimpan data mahasiswa.

Mengatur nilai atribut (setter).

Mengambil nilai atribut (getter).

Menampilkan data mahasiswa melalui method tampilkanData().

 Class Mainprogram

Class Mainprogram adalah class yang memiliki method main() dan menjadi tempat program dijalankan.

Fungsi dari class ini:

Menampilkan menu pilihan ke pengguna.

Menerima input dari keyboard (nama, nim, jurusan).

Membuat objek Mahasiswa baru.

Menyimpan atau menampilkan data mahasiswa.

Mengatur alur program menggunakan switch dan do-while.

Atribut/variabel penting di dalam Mainprogram:

Scanner sc
Digunakan untuk membaca input dari keyboard.

Mahasiswa data
Objek untuk menyimpan 1 data mahasiswa yang dimasukkan user.

int pilihan
Menyimpan pilihan menu yang dipilih user (1, 2, atau 3).
